# Capturas para la tienda

Coloca aquí capturas PNG de la extensión (cualquier tamaño).

Luego ejecuta:

```bash
npm run store:screenshots
```

Se normalizan a **1280×800** (sin alpha) en `store-assets/cws-ready/`.

Si la carpeta está vacía, `npm run store:screenshots` genera vistas previas de marketing automáticamente.
